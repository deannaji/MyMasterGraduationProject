                                                         Assessment Report Generation Software

Welcome, this is my graduation project for my Master degree in CS. It is a web app for generating assessment reports for faculty members of universities and colleges.
This software is done using:
1- Front-end: HTML5, CSS3, JavaScript, and JQuery. 
2- Back-end: Node.JS, Express.JS(MVC)
3- Database: MySQL


For more information, please refer to project documentation inside the documentation folder.
